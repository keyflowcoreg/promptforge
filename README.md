# PromptForge

200+ production-grade AI prompts for Claude, GPT, Gemini.

[![Live Demo](https://img.shields.io/badge/Live_Demo-promptforge.vercel.app-blue?style=for-the-badge)](https://promptforge-indol.vercel.app)

[Live Demo](https://promptforge-indol.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/promptforge/issues)

## Features

- 200+ curated prompts across 9 categories
- Search, browse, and filter by use case
- Free tier with starter prompts, premium unlock for full library
- Covers coding, writing, marketing, data analysis, and more
- x402 crypto payments (USDC on Base)

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion · x402 Payments

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4001](http://localhost:4001) to view it locally.
